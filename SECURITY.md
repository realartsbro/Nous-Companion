# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Nous Companion, please report it privately to:

**Email:** will.hood@protonmail.com

Please include:
- A description of the vulnerability
- Steps to reproduce
- Affected versions

We'll acknowledge your report within 48 hours and provide a timeline for a fix.

## Scope

Security concerns include but are not limited to:
- Path traversal in character import/export
- Unintended file system access
- API key exposure through logs or network calls
- Remote code execution through crafted archives

## Supported Versions

| Version | Supported |
|---------|-----------|
| v0.1.x  | ✅ |

This is a community project. Security fixes will be prioritized and released as patches.
